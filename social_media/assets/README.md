# Assets for social posts

Store images/videos used by posts here, grouped by week to match `posts/weekly/YYYY-MM-DD/`.

Examples:
- `assets/weekly/2025-11-17/POST-000001-eating-photo-1.jpg`
- `assets/weekly/2025-11-24/POST-000002-journey-card-cover.png`

Tip: Use the post ID in file names so assets remain discoverable later.
